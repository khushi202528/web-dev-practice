lets learn about color

Color can be represented as folooawing ways

  1. colour name
  2. hexa decimal
  3. red blue green
  4. RGBA
  5. HSL

COLOR NAME

In this we just write name of colour
Syntax

h1{
    color: red;
  }

HEXA DECIMAL

In this we use to represt code a specific code which looks like

#ff0000

6 hexadecimal digit

#RRGGBB we call it HEX

SYNTAX
p{
   color: #RRGGBB;
   }

RGB

RGB stands for redgreenblue

Syntax
div{
    color: rgb(255,0,0);
}

RGBA

RGBA (red,green,blue nad alpha)
alpha is for transeperancy

Syntax

div  {
     color: rgba(255, 0, 0, 0.5);      /* Semi-transparent red */
}

HSL

hue,saturation,lightness

SYNTAX

span  {
    color:   hsl(0, 100%, 50%);   /* Red */ }
